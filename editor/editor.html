Visual Math Input

Type these LaTeX commands followed by a space to insert symbols:

\alpha, \beta, \gamma...
→ α, β, γ (Lowercase Greek)
\Gamma, \Delta, \Omega...
→ Γ, Δ, Ω (Uppercase Greek)
\pm
→ ± (Plus/Minus)
\times
→ × (Multiply)
\div
→ ÷ (Divide)
\leq, \geq, \neq
→ ≤, ≥, ≠ (Inequalities)
\approx, \equiv, \sim
→ ≈, ≡, ~ (Relations)
\infty
→ ∞ (Infinity)
\forall, \exists
→ ∀, ∃ (Quantifiers)
\in, \notin, \subset
→ ∈, notin;, ⊂ (Set Theory)
\cup, \cap
→ ∪, ∩ (Union, Intersection)
\rightarrow, \leftarrow
→ →, ← (Arrows)
\ldots, \cdots
→ ..., ⋯ (Ellipses)
\sin, \cos, \tan
→ sin, cos, tan (Trig Functions - type with `()` for arguments)
\log, \ln
→ log, ln (Logarithms)
\text{your text}
→ Inserts 'your text' as normal text.

For structures:

/ or \frac
→ Fraction template
^
→ Superscript
_
→ Subscript
\sqrt
→ Square root template
\sum, \int, \lim
→ Summation, Integral, Limit templates

Tip: Arrow keys help navigate within structures.
